iGPTOOLS

A set of utilities for converting waypoints and routes for iGPSPORT BiNavi and BiNavi Air devices.

GPX POI to Location Points

Converts waypoints from a GPX file into the iGPSPORT location points format. Upload a GPX file, download the generated location.dat and point_name_list.dat files after conversion, then copy them to the iGPSPORT/System folder on your device and replace the existing files.

Location Points to GPX POI

Converts iGPSPORT location points into a GPX file. Select the location.dat and point_name_list.dat files from the iGPSPORT/System folder on your device and download the generated GPX file after conversion.

Both device files are required so the tool can restore waypoint names and coordinates.

GPX to CNX

Converts a GPX route into CNX format for iGPSPORT devices. Upload a GPX file, edit waypoints if needed, then download the generated CNX file and copy it to the iGPSPORT/Routes folder on your device.

CNX Point Editor

Allows editing waypoint points inside a CNX route. Upload a CNX file, edit the points, download the updated file, and copy it back to the iGPSPORT/Routes folder on your device.
