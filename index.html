iGPTOOLS

A set of utilities for converting waypoints and routes for iGPSPORT BiNavi and BiNavi Air devices.

GPX POI to Location Points

Converts waypoints from a GPX file into the iGPSPORT location points format. Upload a GPX file, download the generated location.dat and point_name_list.dat files after conversion, then copy them to the iGPSPORT/System folder on your device and replace the existing files.

Location Points to GPX POI

Converts iGPSPORT location points into a GPX file. Select the location.dat and point_name_list.dat files from the iGPSPORT/System folder on your device and download the generated GPX file after conversion.

Both device files are required so the tool can restore waypoint names and coordinates.

GPX to CNX

Converts a GPX route into CNX format for iGPSPORT devices. Upload a GPX file, all tracks and track segments will be merged into one CNX route, edit waypoints if needed, then download the generated CNX file and copy it to the iGPSPORT/Courses folder on your device.

GPX Cleaner

Clean your GPX file and keep only POIs, routes/tracks, and elevation data.

CNX Point Editor

Allows editing waypoint points inside a CNX route. Upload a CNX file, edit the points, download the updated file, and copy it back to the iGPSPORT/Routes folder on your device.

Updating Maps on iGPSPORT BiNavi

  1. Download the latest maps for your region from iGPSPORT BiNavi Map Download.
  2. Extract the archive. Inside you will find two folders:
    • Maps - map files;
    • Router - routing files.
  3. Turn off the device and connect it to your computer or OTG-enabled smartphone.
  4. Open the Maps folder from the extracted archive and locate the files for your region.

    Example:

    RU05002511203HE0EV0YA1OU.map

    Here, RU05 is the region code and 3HE0EV0YA1OU is the geo code.

  5. Open the iGPSPORT/Maps folder on the device and find the files with the same region code (RU05 in this example) and a geo code at the end of the filename.
  6. Create a backup of these files, delete them from the device, and copy the new files from the archive's Maps folder.

    Warning: Do not delete or replace region files that do not contain a geo code. For example, RU05A0260220.map should not be modified.

  7. Open the iGPSPORT/Router folder on the device, create a backup of the existing files, then copy the files from the archive's Router folder and allow them to overwrite the existing files.
  8. Safely disconnect the device and power it on. If any issues occur, restore the files from your backup.
